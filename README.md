# Дипломный проект

Дипломный проект по созданию бэкендов для мобильных приложений. 

## Инструкция по запуску:
1. Запустить Docker.
2. Открыть консоль.

3. В корне проекта (где расположен docker-compose.yaml) ввести команду: 

    ```
    docker-compose up -d
    ```

4. Подождать инициализации сервисов.
5. Воспользоваться ссылками, перечисленными ниже.

## Ссылки на Swagger UI:
